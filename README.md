# Responsive Navbar

## Description

A clean and lightweight responsive navigation bar built using **HTML**, **CSS**, and **Vanilla JavaScript**.  
The project demonstrates a mobile-first navigation system with a hamburger menu that smoothly toggles a mobile navigation panel on smaller screens.

This project focuses on fundamental front-end concepts without using any frameworks or libraries.

---

## Features

- Responsive navigation layout
- Desktop navigation links
- Mobile hamburger menu
- Animated hamburger icon transformation
- Toggleable mobile dropdown menu
- Auto-close mobile menu when a link is clicked
- Pure HTML, CSS, and JavaScript implementation
- No external dependencies

---

## Working

### 1. Desktop View

- Navigation links are displayed inline.
- Call-to-action button remains visible.
- Hamburger menu is hidden.

### 2. Mobile View

- Navigation links are hidden using media queries.
- Hamburger icon becomes visible.
- Clicking the hamburger button:
  - Toggles menu visibility
  - Animates hamburger into a close icon
  - Displays mobile navigation panel

### 3. Menu Interaction

- Selecting any mobile navigation link:
  - Closes the mobile menu
  - Resets hamburger icon state
